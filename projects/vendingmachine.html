Vending Machine

Heavy C++ Programming

Project Overview

This project involved developing a C++ front-end simulation of a vending machine for a fictional pastry company. The primary focus was on implementing robust programming techniques, clean code, and dynamic memory management while following modern C++14 standards.

Scope & Constraints

  • The program simulates a functional vending machine, supporting product display, purchases, stock management, coin handling, and administrator operations.
  • All input and file data (stock, coins) are validated for correctness and edge cases, including malformed files, invalid delimiters, negative values, and empty input.
  • Dynamic memory is carefully managed, with proper allocation and deallocation for all operations to ensure memory safety, verified with Valgrind.
  • Operates entirely in a console environment with persistent storage via structured files, replicating realistic vending machine workflows.
  • Designed to be modular and extensible, allowing future enhancements (e.g., new products, additional menu options) without affecting existing functionality.

Key Objectives & Skills Demonstrated

  • Core C++ Concepts: Implementation of Abstract Data Types (ADTs), linked lists (singly, doubly, and 2D), pointers, and dynamic memory management.
  • File Processing: Loading and saving inventory and coins using structured file formats.
  • Program State Management: Tracking vending machine and transaction states for correct operation.
  • Exception Handling & Validation: Handling invalid input and ensuring program stability.
  • Testing & Debugging: Creation of test cases and verification of memory safety using Valgrind.
  • C++14 and STL Usage: Leveraging modern language features and standard library utilities for efficiency.
  • Design Patterns: Implementing the Command pattern for menu operations to enhance modularity and scalability.
  • Teamwork and Collaboration: Working effectively in a group environment using Git for version control and project coordination.

The project was built entirely from scratch, emphasizing modular design, proper coding style, and adherence to software engineering principles, resulting in a fully functional and maintainable vending machine simulation.

Personal Contributions

As a key member of the development team, I was responsible for:

  • Implementing the core vending machine logic and transaction handling.
  • Designing and implementing custom linked list data structures for inventory management.
  • Developing the file processing module to handle inventory and coin data persistence.
  • Creating robust error handling and validation mechanisms to ensure program stability.
  • Writing comprehensive test cases and debugging code using Valgrind for memory safety.
  • Collaborating with team members to integrate all components into a cohesive system.
