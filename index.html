FriendChise V2 — Live

A multi-tenant task management SaaS for franchise businesses. Built for and actively used at a real Walker's Doughnuts franchise location — real users, real operational data, real stakes.

  • Multi-tenant franchise hierarchy — brand → franchisee org model, org-scoped queries throughout, roles cloned on join, tasks published to and selectively inherited from the franchise network
  • RBAC on every surface — custom roles with 6 permission types; guards on every server action and API route; Upstash Redis sliding-window rate limiting on auth + API routes
  • Demo Mode isolated entirely in code — no separate DB or env flag; demo accounts identified by email suffix, hard caps enforced per session, data auto-deleted within 24 hours
  • Timetable + Task Library — weekly/daily calendar, templates with frozen task snapshots, threaded comments & voting, role eligibility filters, card/list views
  • 582 automated tests — Vitest unit + integration (real Postgres) + 21 Playwright E2E; GitHub Actions CI gates every merge; Sentry error monitoring + session replay in prod
